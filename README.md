# SAS-Padaria

## MVP - Meu App Padaria e suas Funcionalidades

### Menu de Cadastro
- Realiza o Cadastro do produto com as descrições do produto:
(Nome, Codigo, Valor Unitario e Estoque).
- O Usuario poderá cadastrar cada produto de acordo com sua seção
expecifica.

### Menu de Venda 
- Realiza a venda do produto que foi cadastrado através do Menu de Cadastro.
- Realiza o Check do Estoque do usuario para aprovar a compra.
- Realiza o Check do Saldo do usuario para aprovar a compra.
- Realiza o debito do saldo  da conta do usuario.
- Realiza a subtração da quantidade do produto no estoque.
- Apresenta o valor total de acordo com a quantidade escolhida pelo usuario.
- Confirma com o usuário se vai ser dado prosseguimento.
- Caso seja confirmado, o programa fará o check do saldo do usuário e aprovará ou não.
- Caso aprovado: O valor é debitado da conta do usuário e o novo saldo é impresso.
- Caso não aprovado: É apresentada uma mensagem de transação não autorizada, o saldo do usuário e é pedido para que retorne quando for resolvido.

#### OBS 
- O programa não permite que o usuario realize a compra do produto com o saldo inferior ao preço total do produto.
- Assim como não permite que o usuario compre um produto em uma quantidade maior que o estoque.

### Financeiro 
- Realiza Credito de saldo para o usuario.
- Realiza Debito do saldo do usuario.

#### OBS
- O credito do saldo para o usuário é adcionado no Programa com o metodo Credito.
- O Programa é inciado chamando o Usuario + o Metodo Menu Principal.
- No futuro, poderá ser introduzida no menu principal, e tudo poderá ocorrer dentro do programa assim como outras funcionalidades que foram pensadas.
- É Apenas um MVP.

Esse proejto foi criado para servi como portifolio de um CRUD com JAVA BASICO.
